# release-manager

Internal tool for managing and publishing releases. Runs CI checks on all pull requests before merging.

## Development

```bash
npm install
npm run build
```

## CI

Pull requests trigger a build and diagnostics check via GitHub Actions. Build output is visible in the Actions tab.

## Bugs & Support

Found a bug or have a question? [Open an issue](../../issues/new) and we'll look into it.
